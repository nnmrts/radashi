### Usage

Formats the given string in pascal case fashion.

```ts
import * as _ from 'radashi'

_.pascal('hello world') // => 'HelloWorld'
_.pascal('va va boom') // => 'VaVaBoom'
_.pascal('helloWorld') // => 'HelloWorld'
```
